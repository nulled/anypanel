DigiPanel Error code numbers and possible solutions to correct them.

Error 100

ssh2_connect failed: Ensure OpenSSH or the SSH package is installed and accessible on your Server.

First, ensure the IP Address you set in DigiPanel is the IP Address of your Server!

Second, ensure that SSH is installed and running on your server. SSH allows remote access to your Server.

Third, ensure that SSH (Secure SHell) is accessible on port 22 of your Server and is not blocked by any firewalls you may have set up. On Ubuntu Servers run the following commands to allow SSH connections. 
            sudo ufw enable
            sudo ufw allow 22
Install SSH on your Server. Depending on which Linux Server you are running will determine how to install SSH.

Ubuntu Server: sudo apt-get install ssh
CentOS Server: yum install ssh

Error 101

ssh2_auth_password failed: Ensure root user has access vai an ssh login.

First, insure you have your correct root user password set for DigiPanel and your server!
If, the above is confirmed correct and DigiPanel still can not access your Server, see below.

/etc/ssh/sshd_config - edit this file to ensure the following options are set. 
            PermitRootLogin Yes
Note: For Ubuntu servers, the root user does not have a password set, after a default install. You need to become the root user, then set a password for it. The first command sudo -s will make you root user. The second command passwd your_password_here will prompt and confirm a new password for root user. After the password has been set, you can issue the command exit to return to your normal user. 
            sudo -s
            passwd your_password_here
            exit

Error 200

Unable to find the file /etc/issue on your Server to determine distribution type and version.

Your distribution install is corrupt or tampered with and a reinstall is recommended. If you are running a Linux distribution that DigiPanel does not support you can ignore this error. DigiPanel supports the top major Linux Distributions such as; Ubuntu Server, CentOS/RHEL and Debian Server.

Error 201

Distribution is not supported by DigiPanel.
